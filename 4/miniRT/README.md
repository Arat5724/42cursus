# miniRT

RayTracing을 구현하는 과제입니다.

구, 평면, 원기둥, 원뿔을 표현할 수 있습니다.

도형의 위치, 크기, 색깔 등을 지정할 수 있고

색깔 대신 텍스쳐를 입히거나, 거울로 표현할 있습니다.

Phong reflection model이 적용되어있습니다. (https://en.wikipedia.org/wiki/Phong_reflection_model)
